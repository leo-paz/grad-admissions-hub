*GraphQL Folder*

* Create your queries under this folder in the appropriate file and export them as functions.
* Import the query function you want to use into the file where you want to call the Query.
* Usually, you import them into a file under Components, formulate the Query object using apolloClient and then call this component where you want to use it.

*Pages Folder*

* Has the different pages that our frontend will have

*index.js*

* Initilizes our ApolloProvider.
